# AI Agent Hook / Automatic Synchronization Detection

## 1. Overview

The AI Agent Hook provides a controlled entry point for automatic synchronization signal detection.

It allows an AI agent to detect possible similarity signals and prepare candidate audit records.

It does not allow the AI agent to produce final verdicts.

## 2. Core Principle

```text
AI detects.
AI prepares.
AI does not judge.

The agent may identify a possible synchronization signal.

The agent may prepare candidate records.

The agent may request human review.

The agent must not declare influence, imitation, theft, legal responsibility, or final classification.

3. Role in the Protocol Stack
v0.1 = Audit Ruler
v0.2 = Case Study Layer
v0.3 = Structure Fingerprint / Trace Layer
v0.4 = Defense Court Protocol Integration
v0.5 = AI Agent Hook / Automatic Synchronization Detection

The AI Agent Hook connects detection to the existing review system.

It does not bypass it.

4. What the Agent May Do

An AI agent may:

detect possible vocabulary overlap
detect possible structural sequence overlap
detect possible metaphor overlap
detect possible dependency-pattern overlap
detect possible transformation-pattern overlap
detect trace gaps
detect AI reconstruction risk
prepare candidate audit records
prepare candidate structure fingerprints
prepare non-verdict summaries
list missing evidence
recommend human review
5. What the Agent Must Not Do

An AI agent must not:

declare theft
assign legal responsibility
claim direct influence without reference-path evidence
treat similarity as proof
treat a structure fingerprint as a verdict
treat candidate classification as final
bypass the Defense Court Bridge
publish accusation-level claims automatically
6. Candidate Classification

The agent may suggest a candidate A-E classification.

Supported values:

A_accidental_sync
B_structural_convergence
C_indirect_influence
D_direct_reference_or_imitation
E_ai_reconstruction
undetermined

However, this must remain candidate-only.

If the candidate classification is D, human review is always required.

If public accusation risk exists, human review is always required.

If evidence is insufficient, the agent should request more evidence or defer action.

7. Detection Signals

The hook supports the following signal types:

vocabulary overlap
structural sequence overlap
metaphor overlap
dependency pattern overlap
transformation pattern overlap
trace gap
AI context dependency
mixed signals

These are signals, not conclusions.

8. Human Review Boundary

Human review is required when:

candidate D classification appears
structural overlap is high
unique metaphor overlap is high
public accusation risk exists
AI reconstruction risk is present
trace evidence is insufficient
legal or policy relevance exists
human review is requested

The hook may route the case toward a Defense Court Bridge when needed.

9. Recommended Workflow
1. Agent detects possible similarity signal
2. Agent records signal as AI Agent Hook
3. Agent prepares candidate Synchronization Audit Record if needed
4. Agent prepares candidate Structure Fingerprint if needed
5. Agent lists missing evidence
6. Agent requests human review if risk threshold is met
7. Defense Court Bridge is created for sensitive cases
8. Human reviewer maintains, revises, or rejects the candidate classification
10. Relationship to Defense Court Bridge

The AI Agent Hook detects and prepares.

The Defense Court Bridge reviews and escalates.

AI Agent Hook = detection boundary
Defense Court Bridge = human review boundary

The hook should never replace the bridge.

11. Relationship to Structure Fingerprint

The hook may create or recommend a candidate Structure Fingerprint when similarity involves:

structural order
metaphor pattern
dependency pattern
transformation pattern
trace references

But the fingerprint remains a map of resemblance, not proof.

12. Relationship to Synchronization Audit Record

The hook may prepare a candidate Synchronization Audit Record.

The final audit record must be reviewed before being treated as stable.

13. Final Principle

Automatic detection is useful.

Automatic accusation is dangerous.

The AI Agent Hook exists to increase visibility without removing judgment from human review.
